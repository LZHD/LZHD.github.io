```
它会死去，
象大海拍击海堤，
发出的忧郁的汩汩涛声，
象密林中幽幽的夜声。
```

```
它会在纪念册的黄页上
留下暗淡的印痕，
就像用无人能懂的语言
在墓碑上刻下的花纹。
```

```
它有什么意义？
它早已被忘记
在新的激烈的风浪里，
它不会给你的心灵
带来纯洁、温柔的回忆。
```

```
但是在你孤独、悲伤的日子，
请你悄悄地念一念我的名字，
并且说：有人在思念我，
在世间我活在一个人的心里。
```

很多时候都是还没开始就是败了，主要是自己懒
