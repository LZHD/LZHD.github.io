## PHP 算数运算符

|运算符|名称|例子|结果|
|:------:|:------:|:------:|:------:|
|+|加法|$x + $y| $x 与 $y 求和|
|-|减法|$x - $y| $x 与 $y 的差值|
|*|乘法|$x * $y| $x 与 $y 的乘积|
|/|除法|$x / $y| $x 与 $y 的商数|
|%|模数|$X % $y| $x 与 $y 的模数|

## PHP 赋值运算符

|赋值|等同于|描述|
|:------:|:------:|:------:|
|x = y| x = y| 右侧表达式为左侧运算数设置数|
|x += y| x = x + y| 加 |
|x -= y| x = x - y| 减 |
|x *= y| x = x * y| 乘 |
|x /= y| x = x / y| 除 |
|x %= y| x = x % y| 模数 |

## PHP 字符串运算符

|运算符|名称|例子|结果|
|:------:|:------:|:------:|:------:|
|.|串接|$txt1 = "hello"; $txt2 = $txt1."world";|现在 $txt2 包含 "helloworld"|
|.=|串接赋值|$txt1 = "hello"; $txt1 .= "world";|现在 $txt1 包含 "helloworld"|
