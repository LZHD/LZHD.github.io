查看服务器端的python 版本

https://www.python.org/ftp/python/2.7.9/
```sh
	python  -V
```

显示：python 2.6.6

1. 下载Python-2.7.9

```sh
	wget https://www.python.org/ftp/python/2.7.9/Python-2.7.9.tgz    --no-check-certificate
```

2. 解压更改工作目录

```sh
	tar zxvf Python-2.7.9.tgz
	cd Python-2.7.9
```

3. 安装

```sh
	./configure
	make all
	make install
	make clean
	make distclean
```

4. 查看版本信息

```sh
	/usr/local/bin/python2.7 -V
```
