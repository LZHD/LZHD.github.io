## PM2使用方法

### 1. PM2安装
```sh
npm install pm2 -g //全局安装
```
### 2. PM2使用
#### 普通使用
```sh
pm2 start bin/www
pm2 start app.js
pm2 start app.py
pm2 start npm -- start
```
#### 进程监控
```sh
pm2 list //
```
